#   JS Runner using Python

A lightweight JavaScript runtime built for **Thunder Hackathon 2.0**.

This project accepts JavaScript code as input and executes it using an embedded QuickJS engine written through Python.

## 🚀 Features

- Execute JavaScript code from stdin
- Capture `console.log` output
- No Node.js required
- Embedded JavaScript engine
- Fast execution

Supports:

- Variables (`let`, `const`)
- Data types
- Conditions (`if`, `else`)
- Loops
- Functions
- Arrow functions
- Arrays
- Objects
- String operations
- Math operations
- JSON
- Array methods

## 🛠 Tech Stack

| Component | Technology |
|---|---|
| Language | Python |
| JS Engine | QuickJS |
| Input | stdin |
| Output | stdout |
